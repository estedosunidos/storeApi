#Store api

1. Clonar el proyecto
2. ```npm install ```
3. clonar el archivo ```.env.template``` y renombralo a ``` .env ```
4. cambiar las variables de entorno 
5. Levatar base de dato
```
docker-compose up -d
```

6. Ejecutar SEED
```
http://localhost:30006/api/seed
```
7. Levanta ``` npm run  start:dev ```
